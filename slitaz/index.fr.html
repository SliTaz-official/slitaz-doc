Documentation (fr)

La version stable de SliTaz GNU/linux est distribuée avec les manuels des outils de la distribution et les Notes de publication. Tous les manuels ainsi que cette page sont directement accessibles depuis le menu documention de la barre des tâches. Sur le Web, vous trouverez une documentation complète via le site communautaire: doc.slitaz.org. Pour de l'aide vous trouverez un support technique gratuit via la liste de discussion ou le forum.

Pour débuter

Tableau de bord SliTaz (tazpanel)

Depuis le printemps 2011 la nouvelle façon de configurer le système se fait par l'outil TazPanel. C'est une interface web/CGI thémable à partir de laquelle vous pouvez contrôler tout le système. Celà inclut la gestion des paquets, l'ajout et le retrait d'utilisateurs, la création de systèmes Lives et bien plus encore. Chaque page vous donne une petite description pour vous aider à gérer votre système SliTaz. Pour accéder au Tableau de bord vous pouvez utiliser l'entrée "Outils système..." du menu ou cette url :

http://tazpanel:82

Connexions réseau

Ouvrez Tazpanel et cliquez l'onglet Réseau. Cela affiche un résumé simplifié de vos interfaces réseau et de leur état. Pour configurer votre périphérique Ethernet, sélectionnez Réseau filaire dans l'onglet Réseau. A partir de là vous pouvez configurer votre interface pour utiliser DHCP ou une adresse statique. Pour configurer vos périphériques sans fil, sélectionnez Réseau sans fil dans l'onglet Réseau. A partir de là vous pouvez sélectionner le réseau auquel se connecter. Quand vous cliquez sur le réseau désiré, la boite de connexion en dessous est remplie. Ensuite, entrez votre 'passphrase' ou laissez le champ vide s'il n'y en a pas. Enfin cliquez le bouton Configure pour valider vos paramètres.

Vous trouverez plus d'information dans le manuel : Configuration du réseau

Applications

Pour avoir la liste de toutes les applications graphiques, vous pouvez utiliser le bouton de menu en haut à gauche de l'écran. Pour avoir la liste de tous les outils en ligne de commande, vous pouvez ouvrir un terminal et appuyer deux fois sur la touche tabulation. Tazpanel fournit une interface graphique pour la gestion des paquets avec l'onglet Paquets. Pour récupérer la liste de tous les paquets installés, ouvrez Tazpanel et choisissez la page Mes Paquets. Pour installer de nouveaux paquets logiciels, vous devez avoir une connexion réseau, le DVD des paquets ou une clé USB avec des paquets précédemment téléchargés. Bien sûr toute la gestion de paquets peut aussi être faite en ligne de commande avec tazpkg.

Bureautique

La base de paquets SliTaz fournit diverses applications de bureautique légères. Tous ces paquets sont facilement installables en quelques clicks par le gestionnaire de paquet de Tazpanel. SliTaz fournit aussi une façon simplifiée pour installer la performante suite LibreOffice : cherchez simplement "LibreOffice" dans le gestionnaire de paquet, double-clicker sur "get-LibreOffice", cocher la case "Installe automatiquement le paquet", et cliquez sur Installer.

Pour avoir une suite bureautique légère : utilisez Abiword comme traitement de texte, Gnumeric comme tableur et Homebank pour gérer vos comptes.

Multimedia

La plupart des saveurs SliTaz incluent une grande variété de pilote de carte son, un contrôle du volume et un lecteur audio léger pour écouter facilement n'importe quel fichier MP3 ou OGG. La carte son est autodétectée au démarrage et le support audio fonctionne directement sur beaucoup de systèmes. Si vous ajoutez manuellement un nouvel utilisateur et qu'il ne peut pas écouter de fichiers audio, veuillez vérifier que l'utilisateur appartient bien au groupe audio. Pour tout autre problème vérifiez la documentation et/ou utilisez le forum.

Les videos utilisant un codec HTML5 habituel comme WEBM ou MPEG4 peuvent être vues directement. Les autres peuvent être vues en installant un simple paquet ou en utilisant un système LiveCD ou LiveUSB. L'équipe de SliTaz à empaqueté quelques uns des meilleurs lecteurs video disponibles sous GNU/Linux : mplayer, xine-ui et vlc.

Devenir Root

Par défaut, le bureau de SliTaz n'est pas lancé par l'utilisateur root : il demande le mot de passe root (pas défaut root) pour chaque action super utilisateur. C'est plus sûr. Pour devenir root dans un xterm, tapez su et entrez le mot de passe root. Pour lancer une application graphique en root, utilisez subox. Par exemple, pour lancer leafpad en root vour lancerez subox leafpad.

Installation

SliTaz peut être installé sur le disque dur ou une clé USB avec l'installeur SliTaz du Tazpanel. L'installeur fait une installation complète sur disque dur. SliTaz ne fonctionnera plus de mémoire (RAM) comme dans les modes Live ou Frugal et tout sera persistant.

Pour générer un système LiveUSB, vous pouvez utiliser "Créer un liveUSB" dans le menu "Outils système..." ou tazusb en ligne de commande. Pour plus d'information, voir le manuel : LiveUSB
