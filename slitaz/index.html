Stable - Releases Notes

[fr] La version stable de SliTaz GNU/linux est distribuée avec les manuels des outils de la distribution et les notes de publication. Tous les manuels ainsi que cette page sont directement accessibles depuis le menu documention du gestionnaire de fenêtres JWM présent sur le LiveCD. Sur le Web, vous trouverez une documentation complète via le livre de main ou Handbook de SliTaz et un support technique gratuit via la liste de discussion ou le forum.

[en] SliTaz GNU/Linux stable release is published with the project tools manuals and the English Release Notes installed on the system. Manuals and this page are accessible through JWM menu. Please, check the English project page for general informations on SliTaz. To use and configure the system you can read the English Handbook.

Cooking - Informations

La version en cuisson ou Cooking ne fournit pas de notes ou autres, elle est trop souvent mise à jour et modifiée. Cela dit, vous pouvez vous référer à la documentation du site. Nous vous conseillons de commencer par consulter le livre de main ou Handbook. Merci de nous faire part de vos suggestions et remarques via les informations de contact se trouvant sur le site web de SliTaz ou via le forum du projet.
