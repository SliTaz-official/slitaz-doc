[en] Releases Notes

SliTaz GNU/Linux stable release is published with the project tools manuals and the English Release Notes installed on the system. Manuals and this page are accessible through the Openbox menu. Please, check the English project page for general information on SliTaz. To use and configure the system you can read the English Handbook. Community documentation can be found on the SliTaz Wiki and support on the SliTaz community Forum.

The Cooking version doesn't provide Release Notes, any changes are announced on the web site and the Labs provide fresh news about the project activity. Like the stable version, you will find documentation on the web site and support on the community forum.

[fr] Notes de publication

La version stable de SliTaz GNU/linux est distribuée avec les manuels des outils de la distribution et les notes de publication. Tous les manuels ainsi que cette page sont directement accessibles depuis le menu documention du gestionnaire de fenêtres Openbox présent sur le LiveCD. Sur le Web, vous trouverez une documentation complète via le livre de main ou Handbook de SliTaz et la documentation communautaire sur le Wiki. Pour de l'aide vous trouverez un support technique gratuit via la liste de discussion ou le forum communautaire.

La version en cuisson ou Cooking ne fourni pas de notes de publication, les chagements sont annonçés sur le site web et les laboratoires proposent des nouvelles fraîches. Comme pour la version stable, vous pouvez vous référer à la documentation du site et trouverez de l'aide sur le forum du projet.
